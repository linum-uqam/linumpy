linumpy
=======

**linumpy** is the main library of the *Laboratoire d'Imagerie Numérique,
Neurophotonique et Microscopie* (`LINUM <https://linum-lab.ca>`_) at
Université du Québec à Montréal. It provides Python tools and CLI scripts
for serial optical coherence tomography (S-OCT) and serial blockface
histology: tile acquisition, preprocessing, stitching, 3-D reconstruction,
stacking, and downstream analysis.
